## Overview

AhasanHub is a modern, multi-page personal brand website built with **Next.js**, **TypeScript**, and **Tailwind CSS**. It represents expertise in Artificial Intelligence, ERP, Cloud, SaaS, Full-Stack Development, Software Architecture, and Digital Transformation Consulting.

The project is designed as a production-ready consulting platform rather than a traditional portfolio website.

---

## Objectives

- Build a strong professional brand
- Showcase services and expertise
- Publish technical insights
- Present case studies and projects
- Generate consulting leads
- Support future digital products and SaaS offerings

## Local Development

### Prerequisites

- Node.js (Latest LTS)
- pnpm

### Install

```bash
pnpm install
```

### Start Development Server

```bash
pnpm dev
```

Open:

http://localhost:3000

### Build Production

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

## AI Development Workflow

- Product Owner: User
- Planning & Architecture: ChatGPT
- Coding Agent: Codex
- Source Control: GitHub
- Deployment: Vercel

Each task should:

1. Be implemented in a dedicated branch.
2. Focus on a single feature.
3. Reuse existing components.
4. Pass linting and build checks.
5. Be reviewed before merging.

---

## Deployment

Recommended platform:

- Vercel

Deployment workflow:

1. Push changes to GitHub.
2. Create a Pull Request.
3. Review changes.
4. Merge into the main branch.
5. Deploy via Vercel.
